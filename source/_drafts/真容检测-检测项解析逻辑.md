### config-配置检测大项的显示

 不同的套餐，具体的检测大项也有可能是不同的。

 截止我写这个文档时，目前的套餐基本大项配置是一样的，只有B端检测报告，是没有亮点配置的。

 在检测详情页面，显示View的时候解析了这个字段。

```
"config": [

        {
            "4s_report": "4S店维保查询记录"
        },
        {
            "evaluate": "车辆估价"
        },
        {
            "appearance": "外观"
        },
        {
            "Interior": "内饰及电器"
        },
        {
            "nacelle": "发动机舱"
        },
        {
            "Chassis_girders": "底盘大梁"
        },
        {
            "Other_items": "其他项"
        }
        {
            "Summary": "总体评价"
        },
        {
            "ResultType": "检测结果"
        }
    ]

```
