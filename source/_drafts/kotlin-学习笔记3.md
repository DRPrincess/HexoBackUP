# Java 的四种修饰符

Java 提供四种修饰符 public ,protected,default,private。其中 default 不是一个关键字，而是没有定义类型时，默认的修饰符。
修饰符的存在，是为了定义类，属性和方法的访问权限。
四个修饰符的访问权限为：
|修饰符&访问权限|类|包|子类|其他包|
|:---:|:---:|:---:|:---:|:---:|
|public|√|√|√|√|
|protected|√|√|√|x|
|default（默认）|√|√|同包内的子类 √ 其他包的子类x|x|
|private|√|x|x|x|
